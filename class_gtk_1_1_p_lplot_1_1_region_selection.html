Detailed Description

a class providing support for selecting a region in a plot

Several plot classes (Plot2D, PlotPolar, PlotContour and PlotContourShades) that represent data in a two-dimensional grid inherit from this class, allowing users to drag a selection box using a mouse gesture (mouse button press, drag, mouse button release). The default signal handler will initiate a redrawing of the plot, with the visible region set to correspond to the initally drawn box. This is shown in Example 1: Plot data properties and zooming and others.

This method takes care of coordinate transformations when using non-linear axes

This is a default handler for signal_select_region()

This function passes the plot (data) coordinates of the selection box to set_region, in order to set the plotted range corresponding to the selection box. If this behavior is not desired, derive the class and implement your own on_select_region method.
