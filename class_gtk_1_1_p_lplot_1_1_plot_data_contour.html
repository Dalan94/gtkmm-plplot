Detailed Description

a class for PlotContour datasets

Instances of this class will hold a single (and the only possible) dataset for a PlotContour plot. Apart from the data necessary to generate a contour plot, it contains several properties that will determine the appearance of the plot. The input data can be provided as std::vector (X and Y), std::valarray (X and Y), PLFLT** (Z, array of arrays) and boost:multi_array (Z). This datatype is demonstrated in Example 7: a Simple contour plot.
