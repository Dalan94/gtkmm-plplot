Detailed Description

Principal plotting widget.

Inheriting from Gtk::DrawingArea, this is by far the most important widget in PLplot, as it will contain the plots and the datasets. Depending on which constructor is chosen, either an empty canvas will be produced, or one containing a single plot. It is possible however, to add any number of plots on the canvas. The positions and dimensions of these plots are properties of the plots, that have to be provided when invoking their constructors. This is demonstrated in Example 3: Adding, removing, showing and hiding plots and Example 4: Displaying multiple plots on a single canvas.
